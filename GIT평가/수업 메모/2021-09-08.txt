// 접근 제한자  - visibility
** public : 모든 곳에서 접근 가능
protected : 외부 패키지에서 사용불가, 이 클래스에서 상속받아서 사용하면 사용가능
     -  extends 클래스명 (상속받는다)
default : 아무것도 적지 않은 것, 기본적으로 외부에서 사용불가
** private : 클래스 내부에서만 사용가능

- 자바에서 기본적으로 필드는 private이라 외부에서 접근가능

*** 기본값
필드는 private
