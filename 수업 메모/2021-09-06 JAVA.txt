자바 첫시간

자바의 특징: 이식성이 높은언어
소스코드파일(.java) -> 한번의 컴파일 바이트 코드 파일생성(.class) -> 맥, 윈도우, 리눅스 모두 사용가능

class 파일은 바로 실행될수있는 파일이 아님.

JVM 이라는 가상머신에서 자바의 class파일이 돌아간다


main: 메인 메서드, 명령어 시작부분

한줄끝에 세미콜론

프로젝트 project : 빌드단위, 애플리케이션 단위
패키지 pakage : 코드 묶음 단위, 하나의 디렉토리(폴더)라고 생각
JRE system Library : 개발할때 쓰는 개발툴 라이브러리

변수는 지역변수, 전역변수로 나뉜다, 

** 원시형 데이터 타입 ( primitive type )
자바스크립트: number, string, boolean
자바: 자바에는 문자열이 기본 데이터 타입이 아님
  number : 
    정수 (소숫점이 없는 것)
      * byte
         - 8비트 (256개), 2를 8번 제곱함, 2^8
         - -128 ~ 0 ~ 127   1바이트
     * char (문자를 담는 공간)
         - A, B, c   2바이트 , 문자를 저장함
         -  0 ~ 2^16
     * int 4byte
         - 21djr
     * long 8byte

    실수 (소숫점이 있는 것)
     * float, double (더 많이 씀)
  논리형
     true, false  = 1byte


byte(1) < short(2) < int(4) < long(8) < float(4) < double(8)
